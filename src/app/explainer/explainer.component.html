Activity app

What you need to build

Using the API documented at this link build an application that displays random activities for the day.

You can use any library you want for this task.

Good UI design is a plus

Good artifacts design & structuring is a big plus

Requirements

Must have

  • Display a single button labeled "Fetch activity" that fetches a random activity from the API and adds it to a list. The list starts empty. Display a skeleton state when a fetch is in progress. Design is up to you.
  • The list must be automatically organized by activity category (use the type property on the fetched activity). Each activity on the list should display only the main title, and display additional fetched details when selected by the user
  • If the same activity is randomly fetched more than once, automatically discard it and fetch a new one
